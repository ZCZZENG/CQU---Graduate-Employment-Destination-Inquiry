# CQUPM Employment Insights

> 基于重庆大学校内公开可查询的历史毕业去向信息，进行匿名聚合与结构分析的学习型项目。

## 项目背景

在当前就业与升学环境快速变化的背景下，学生在进行学业与职业规划时，往往只能依赖零散经验或非系统性信息。
本项目尝试基于校内 **公开可查询** 的历史毕业去向数据，对其进行结构化整理与可视化分析，
以帮助学生更直观地理解不同届次、不同学历层次的去向选择结构。

## 数据来源与边界说明

- 数据来源：重庆大学就业系统中 **学生端可查询的历史毕业去向汇总信息**
- 数据处理方式：
  - 不包含任何个人信息（如姓名、学号、联系方式等）
  - 仅使用匿名聚合后的统计结果
  - 对低频、不可识别去向统一归为“其他”
- 本项目 **不提供、也不存储原始个人级数据**

## 方法论说明（简要）

- 分析对象：机械学院 2021–2023 届毕业生
- 分学历层次处理：
  - 本科、硕士作为主要分析对象
  - 博士因样本量较小，单独整理
- 去向筛选规则：
  - 仅统计在至少两个不同毕业届次中，且单届不少于 2 人的单位
  - 其余去向统一归为“其他”，以降低偶然性噪声
- 单位名称处理：
  - 高校名称采用精确匹配
  - 企业名称采用模糊匹配（合并集团 / 子公司等差异）

## 使用说明

本仓库内容仅用于：
- 就业与升学结构分析示例
- 数据分析与可视化学习
- 就业认知与决策参考

不用于任何形式的排名、评价或考核。
